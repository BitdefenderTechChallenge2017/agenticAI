Agentic Code Review

This repo holds the source code for a team of AI agents that spring into action and perform a code review each time a .py or .js file in the "source" directory is changed or added. One agent is an expert in code security. Another is an expert in debugging, and a third is an expert in code optimization. A lead agent examines each source-code file and hands them off to the other agents. Check out handle_changed_files.py in the "scripts" directory to see the source code.

To see these agents in action:

  1. Create a GitHub repo and copy these files into it

  2. Create a GitHub secret named API_KEY and set it equal to your OpenAI API key.

  3. Check a .py or .js file into the "source" directory (or change an existing one).

  4. Wait a moment for the agents to run. The look in the "reports" directory for an MD file containing a code review.

After adding or changing a file, you can open the repo in your browser and navigate to the "Actions" tab to see the script run in real time. handle_changed_files.py is triggered by a GitHub action.
